# Almere D6 deep-research learnings

## Summary
Current sprint: Sprint 31.5 - Deep-research bronintake en top-layer verificatie.

This note collects what the repository learned from the Sprint 31.5 deep-research intake. It separates source-backed public learning from remaining validation questions. Deep-research reports are not evidence; only selected public sources that were ingested through the pipeline count as source material.

Current top-layer position:

| Metric | Value |
| --- | ---: |
| D6 responsibility rows | 12 |
| Rows with public-source prefill | 11 |
| Rows marked `settled` | 0 |
| Rows still needing human validation | 12 |
| Named D6 public-search gaps remaining | 0 |

## What Changed

Six deep-research intakes were processed:

| Intake | Main effect |
| --- | --- |
| Healthy School | Added Almere/GGD/JGZ school and prevention context. Gezonde School moved from weak public basis to source-backed validation candidate. |
| Digital and operational infrastructure | Added public sources for RTP, RSO/data infrastructure, thuismonitoring, GGD monitoring and ACP/PZP data-sharing use cases. |
| Informal support and social-base access | Added public sources for buurtontmoeting, wijkbudget, De Schoor, VMCA, Humanitas, mantelzorg and resident-initiative support. |
| PGA / Zorgzaam / Flever interface | Clarified the public role split between PGA, Zorgzaam Flevoland, Flever, GGD Flevoland and IZA/PGA context. |
| Samen Sterker in de Wijk | Strengthened public evidence for Samen Sterker as an active mental-health neighbourhood initiative with municipal, regional, ZonMw, PGA and project-site context. |
| D6 settlement evidence audit | Confirmed the public-source boundary: public evidence is now strong enough for validation preparation, but not for settlement of D6 classification, ownership, mandate or funding. |

## Learned Per D6 Component

| D6 component | What we learned from public sources | Source-backed interpretation now possible | What this changed |
| --- | --- | --- | --- |
| Inloopvoorzieningen sociaal en gezond | Almere has public municipal subsidy routes for buurtontmoeting, wijkbudget, wijkteam social-contact access, De Schoor buurtkamers and social-base policy/advice context. | These are real public building blocks for validating D6-related inloop and social-base infrastructure. | This is no longer a broad public-source gap; it is now a classification, ownership and funding validation question. |
| Wijkteams Almere | Public Wijkteams Almere material supports wijkteams as an existing local support/access structure. | Wijkteams are a plausible validation candidate for local D6 access and social infrastructure. | The baseline existence question is answered publicly; formal D6 status is not. |
| Stevige Lokale Teams | Public council and Documentwijzer/Notubiz material show a local SLT decision route and a link to JGZ/wijkteams. | SLT is highly relevant to D6 local-team validation. | The repository can point to a real public decision route, but cannot treat it as a full D6 settlement. |
| JGZ Almere | Public GGD/JGZ sources show a broad 0-18 youth-health role, with relevance to families, schools and local support. | JGZ Almere is a strong actor to validate for D6/JGZ and SLT-related roles. | The role is no longer only inferred from national D6 logic; it has Almere/GGD public-source support. |
| GGD Flevoland-coordinatie | GGD budget, Kadernota and GALA-region material support GGD Flevoland's regional public-health, prevention, knowledge, advice, JGZ and monitoring relevance. | Some D6 coordination and monitoring functions may sit naturally at GGD Flevoland scale. | The question narrowed from “is GGD relevant?” to “which GGD tasks are D6-specific for Almere?” |
| Gezonde School / mentale gezonde school | Almere and GGD sources now support local school wellbeing/prevention context, GGD Gezonde School advisers, JGZ linkage and LEA partner context. | Gezonde School / mental-health school prevention is a source-backed validation candidate. | This row gained public-source prefill; it is no longer an empty or generic school-prevention gap. |
| Kennis & Advies / monitoring / dashboards | GGD Kennis & Advies sources show monitors, dashboards, epidemiology, data analysis and advice functions. | GGD Kennis & Advies is a candidate monitoring/data actor for D6 validation. | The public monitoring capability is clear; the D6 steering arrangement remains open. |
| Samen Sterker in de Wijk | Public sources show active Samen Sterker implementation context through Almere, Zorglandschap Wmo, ZonMw, PGA, subsidy trace, evaluation context and project-site material. | Samen Sterker is a strong validation candidate for D6-related mental-health wijkinfrastructure. | This row is now much stronger publicly, but still not settled as D6 infrastructure. |
| PGA / Zorgzaam Flevoland-interface | Public sources clarify a role split: PGA as local transformation programme, Zorgzaam Flevoland as regional IZA/regioplan movement, Flever as participation/support actor, and GGD as monitoring/advisory partner. | The interface can now be described as a public-source-backed programme/governance interface needing validation. | The repository can separate actors better and avoid collapsing PGA, Zorgzaam, Flever and GGD into one actor. |
| Digitale en operationele infrastructuur | Public sources support distinct components: RTP, RSO/data infrastructure, thuismonitoring, GGD monitoring and ACP/PZP data-sharing pilots. | Digital/operational infrastructure is not one public object; it is a set of validation candidates. | The question moved from finding examples to classifying and governing subcomponents. |
| Burgerinitiatieven en informele steun | Public sources show wijkbudget, resident-led neighbourhood meeting support, De Schoor, VMCA, Humanitas, mantelzorg and prevention-network context. | The repository now has a public-source-backed partner and mechanism map for validation. | This row is no longer a missing-inventory problem at public level; it is a D6 classification and funding question. |
| Financiering en budgetafbakening | National funding architecture, work-agenda process and some local subsidy/funding traces are visible. | The repository can warn about double-counting risk across AZWA-D6, D5, SPUK/GALA, PGA, regular municipal budget and Zvw/insurer funding. | Public sources support the need for a funding matrix, but do not fill that matrix. |

## Main Repository Learning

The public-source corpus is now strong enough to prepare a validation conversation. It is not strong enough to produce final bestuurlijke D6 wording.

The practical shift is:

- Before deep research: several rows had weak or generic public support.
- After deep research: most rows have concrete public-source prefill.
- Still unchanged: no row is `settled`; formal D6 classification, owner, mandate and funding require validation.
