# Almere D6 stakeholder information request

## Summary
Current sprint: Sprint 25.5 - D6 stakeholder validation.

This request pack lists the information needed from stakeholders after public-source intake. It is not a final report and not a policy recommendation. It should be used to conduct validation sessions and to update `docs/review/almere_d6_stakeholder_validation_log.md`.

## What Public Sources Can Already Support

Public sources can support a validation-ready structure for D6 in Almere:

- National D6 basisinfrastructuur themes.
- Public evidence for Wijkteams Almere as local access/support structure.
- Public council route for Stevige Lokale Teams and the 19 February 2026 amended/adopted proposal.
- Public evidence for GGD Flevoland, JGZ Almere, Kennis & Advies, monitoring and prevention roles.
- Public implementation context for Samen Sterker in de Wijk and PGA/Zorgzaam/Flever.
- National frameworks for local teams and access.

Public sources do not yet settle local ownership, mandate, D6 classification, funding allocation or safe work-agenda wording.

## Evidence Rule

No D6 register row may move to `settled` unless the validation answer is traceable to either:

- an authoritative public/local source already in the corpus;
- a new source added to the corpus;
- a stakeholder validation log entry with role, organization, rows discussed, confirmed points and validation evidence path.

If stakeholders disagree, use `review_needed`. If evidence remains indirect, use `inferred`. If owner or funding remains unidentified, use `unknown` or `review_needed`.
