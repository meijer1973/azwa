## Standard Row Questions

Ask these questions for each D6 register row:

1. Is this D6 component correctly described?
2. Does this Almere provision actually exist as described?
3. Is it formal D6 infrastructure, adjacent infrastructure, candidate infrastructure, or only context?
4. Who owns it?
5. Who executes it?
6. Who coordinates it?
7. Which partners are necessary?
8. Which scale applies: Almere, IZA/AZWA Flevoland, GGD Flevoland, PGA/Zorgzaam Flevoland, zorgkantoor or other?
9. Which funding source applies?
10. Is there a formal decision?
11. Which document or validation record proves that?
12. Can the row move to `settled`?
13. What wording is safe for the work agenda?
14. What wording is not yet safe?
