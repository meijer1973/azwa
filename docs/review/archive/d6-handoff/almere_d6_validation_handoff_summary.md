## What Policymakers Need To Validate

- Which D6 components are formal D6 infrastructure, adjacent infrastructure, candidate infrastructure or context.
- Who owns, executes and coordinates each component.
- Which scale applies: Almere, IZA/AZWA Flevoland, GGD Flevoland, PGA/Zorgzaam/Flever, zorgkantoor or other.
- Which funding source applies per component and where double counting is a risk.
- Which wording is safe for later work-agenda drafting.

## Evidence Needed Back

Each answer should point to one of:

- public/local source;
- internal/local document;
- decision record;
- finance/controller confirmation;
- completed validation-log entry with role, organization, rows discussed and evidence path.
